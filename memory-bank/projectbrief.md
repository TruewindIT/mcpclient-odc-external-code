# Project Brief: MCPClient OutSystems External Library

**Goal:** Create and maintain an OutSystems External Library (`MCPClient`) that enables OutSystems applications to interact with Model Context Protocol (MCP) servers using the Server-Sent Events (SSE) protocol.

**Core Functionality:**
*   Connect to MCP servers via SSE.
*   List available tools on a server.
*   Call specific tools on a server, passing arguments with appropriate data types.
*   Provide integration points for OutSystems AI Agent Builder.
*   Package the library for deployment in OutSystems environments.
